Módulo 3: Comienza a construir tu app

En este módulo comenzarás a construir tu app usando el Desktop Modeler. Aprenderás a crear y editar páginas y botones, y a definir la estructura de datos con el Domain Model.

La siguiente fase: Construyendo tu app

Después de definir las historias de usuario y crear el wireframe, es momento de desarrollar la app. El primer paso es revisar y priorizar las historias de usuario para comenzar por la que más valor aporta.

Ejemplo: Para Jimmy, la historia "Como administrador, quiero poder acceder y gestionar mis datos fácilmente para administrar mi empresa de forma eficiente" es ideal para empezar.

  • Tener una página principal que dé acceso a otras páginas.
  • Ajustar el diseño de la página principal para los botones.
  • Agregar los botones.
  • Agregar las páginas a las que los botones enlazarán.
  • Enlazar los botones a las páginas que muestran la información.

Desktop Modeler

Usarás el Desktop Modeler para construir la app de Jimmy. Con esta herramienta puedes crear páginas, definir la estructura de datos y acceder a funciones avanzadas de Mendix.

Abrir tu app en el Desktop Modeler
  1. En el Developer Portal, haz clic en la flecha junto a Edit App y selecciona Edit in Desktop Modeler.
  2. Si no lo tienes instalado, se te ofrecerá descargarlo. Si tienes varias versiones, selecciona la correcta.
  3. Inicia sesión con tus credenciales Mendix.
  4. La app se descargará automáticamente del Team Server.
Explorando el Desktop Modeler
  • Barra de control superior: menús y botones para abrir, editar, ver, ejecutar y desplegar la app.
  • Project Explorer (izquierda): acceso a recursos como páginas, microflows y navegación.
  • Panel derecho: Toolbox, Propiedades y Conector.
  • Pestañas inferiores: Historias, Cambios y Errores.
  • Centro: Editor para construir recursos, con modos de edición y vista previa.
Crear la página principal
  1. Elige el layout Atlas_Default para mantener consistencia.
  2. Cambia el título a "LearnNow Training Management" y agrega un subtítulo.

Estructura y Layout Grid

El layout grid te permite estructurar tus páginas en filas y columnas, adaptando el diseño para desktop, tablet y móvil. Puedes usar containers para agrupar widgets y darles estilo o alineación.

Ejemplo de información a mostrar:
  • Cursos
  • Lugares
  • Profesores
  • Alumnos

La página principal debe tener cuatro botones, adaptados para cada dispositivo (responsive).

Agregar columnas
  1. Elimina el layout grid existente y agrega uno nuevo con cuatro columnas (3,3,3,3).
  2. Configura las clases para que en tablet se muestren dos columnas por fila.

Agregar botones
  1. Usa el building block Card Action en cada columna.
  2. Cambia el texto y el icono de cada botón (Cursos, Lugares, Profesores, Alumnos).
Crear páginas
  1. Crea páginas de tipo List_Default para cada entidad: Course_Overview, Location_Overview, Teacher_Overview, Trainee_Overview.
  2. Enlaza cada botón a su página correspondiente usando la acción "Show a page".

Agregar datos y definir el modelo de dominio

El Domain Model es la representación visual de la estructura de datos. Cada entidad será una tabla en la base de datos y cada atributo una columna.

Ejemplo de entidades y atributos:
  • Course: Título (String), Descripción (String), Duración (Integer), Precio (Decimal)
  • Location: Nombre (String), Dirección (String)
  • Teacher: Nombre (String), Email (String)
  • Trainee: Nombre (String), Dirección (String), Email (String)

Usa CamelCase para nombrar entidades y atributos.

  1. Abre el Domain Model y crea las entidades y atributos necesarios.

Obtener la última versión de la app LearnNow

Si necesitas empezar desde cero o actualizar tu proyecto, puedes importar el paquete del proyecto o del módulo desde la pestaña Recursos.

Importar proyecto:
  1. Descarga el paquete desde Recursos.
  2. En Desktop Modeler, selecciona File > Import Project Package.
  3. Sube el proyecto al Team Server.
Importar historias de usuario:
  1. Descarga el archivo Excel desde Recursos.
  2. Importa las historias en la sección Collaborate > Stories.
Actualizar módulo:
  1. Descarga el módulo desde Recursos.
  2. Importa el módulo y reemplaza el existente.
  3. Configura la navegación para enlazar las páginas principales.
